Design your product gallery thumbnail easily.

== Description ==

This plugin is for change product gallery layout. horizontal and vertical thumbnail gallery and provide inner and outer zoom effect for gallery images.

== Frequently Asked Questions ==

= Is this plugin support all theme ? =
Yes, it's support with all themes but need some design changes for perfect looks. It's work on woocommerce default class.
